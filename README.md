# SBOS — SKULL Business Operating System

Sistema operativo empresarial soberano.

## Daemons
- `bos/` — Control plane: instala, vigila, repara
- `bkernel/` — Bus de datos vía PostgreSQL WAL
- `biedata/` — Integración tributaria (SIN/AFIP/SAT)
- `bcompass/` — IA soberana (Ollama + Qdrant)
- `bsearch/` — Búsqueda federada soberana
- `bauth/` — Identidad BitMask 64-bit
- `bhnexus/` — Hardware + WebSocket broker
